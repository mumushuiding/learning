# 第11章 基于对象编程

## 11.1 抽象编程 (Abstract Programming)

抽象：行为抽象和数据抽象

### 11.1.1 行为抽象 (Action Abstract)

### 11.1.2 数据抽象 (Data Abstract)

数据：用以描述具体事物的性质。

数据类型：数据+操作

数据抽象：具体事物描述的一个概括

抽象数据类型：复合数据类型

### 11.1.3 数据结构 (Data Structure)

数据结构：按一定逻辑结构组织，并带有自身操作的数据，基本等于数据类型概念

描述群体数据（容器）的抽象数据才是数据结构的真正体现

描述抽象数据类型和算法的编程就是基于对象的编程。
